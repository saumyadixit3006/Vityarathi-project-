Habit consistency tracker
HABIT CONSISTENCY TRACKER
  
                      It is a simple Python- based project that helps users track their daily habits,calculate consistency scores and predict next week'sperformance based on past data.
  OVERVIEW
  The habit Consistecy Tracker is a begginner - friendly  command -line appplication developed using pyton.
  It allows us to :
  Record whether they completed a habit each day.
  Calculate a 7 - day consistency score
  Predict next week's consistency score using a simple trend - based formula.
  Store data automatically in JSON file 

  PROJECT OBJECTIVES
  Identify a real-world need:daily habit tracking.
  Apply python fundamentals.
  Use modular code and functions.
  Implement file handling.
  Perform basic trend prediction.
  provide clean output for evaluation.

  FREATURES
  
  Enter day +"yes/no" for habit completion.
  Shows your performance for the last 7 days.
  Uses a simple trend :
  if consistency increasing - boost
  if decreasing- reduce
  otherwise - stable 
  score range : 0-100
  Exit Program
